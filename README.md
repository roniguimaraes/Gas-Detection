# Frappe Web Socket Project

## Overview
This project demonstrates a web application built using Python for the backend, where gas detection with YOLO occours, and a modern frontend framework. It showcases real-time communication capabilities through WebSocket integration.

## Setup Instructions
1. **Backend Setup**
   - Install dependencies: `pip install -r backend/requirements.txt`
   - Run the server: `python backend/app.py`

2. **Frontend Setup**
   - Install dependencies: `npm install`
   - Start development server: `npm start`
